Microsoft Project Calendar Tutorial

Step-by-step guide to creating and using a calendar

Objective

This guide explains how to create a new calendar, modify working time, and apply it in Microsoft Project.

Step 1: Create a New Calendar

  1. Go to Project tab
  2. Click Change Working Time
  3. Select Create New Calendar
  4. Give your calendar a name (e.g., My Calendar)
  5. Click OK

Step 2: Change Working Hours

  1. Select a day (e.g., Monday)
  2. Click Details…
  3. Set working hours (e.g., 09:00–12:00, 13:00–17:00)
  4. Click OK

Step 3: Add Special Days (Holidays)

  1. Go to Exceptions tab
  2. Add a holiday (e.g., Christmas)
  3. Select the date range
  4. Click OK

Step 4: Use the Calendar in a Project

  1. Go to Project Information
  2. Select your new calendar from the dropdown
  3. Click OK

How the Calendar Affects Scheduling

Once applied, the calendar controls task scheduling:

  • Tasks will only be scheduled during working hours
  • Non-working days (weekends, holidays) are skipped
  • Task durations adjust automatically if working hours change
Auto Scheduled tasks recalc automatically based on the calendar.
