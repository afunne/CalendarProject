Microsoft Projecti kalendri juhend

Samm-sammuline juhend kalendri loomiseks ja kasutamiseks

Eesmärk

See juhend selgitab, kuidas luua uus kalender, muuta tööaega ja rakendada seda Microsoft Projectis.

Samm 1: Uue kalendri loomine

  1. Mine Project vahekaardile
  2. Klõpsa Muuda tööaega
  3. Vali Loo uus kalender
  4. Anna oma kalendrile nimi (nt Minu kalender)
  5. Klõpsa OK

Samm 2: Tööaja muutmine

  1. Vali päev (nt esmaspäev)
  2. Klõpsa Detailid…
  3. Määra tööajad (nt 09:00–12:00, 13:00–17:00)
  4. Klõpsa OK

Samm 3: Eripäevade (pühade) lisamine

  1. Mine Erandid vahekaardile
  2. Lisa püha (nt jõulud)
  3. Vali kuupäevavahemik
  4. Klõpsa OK

Samm 4: Kalendri kasutamine projektis

  1. Mine Projekti teave
  2. Vali oma uus kalender rippmenüüst
  3. Klõpsa OK

Kuidas kalender mõjutab ajakava

Kui kalender on rakendatud, kontrollib see ülesannete ajastamist:

  • Ülesandeid ajastatakse ainult tööaegadel
  • Töövälised päevad (nädalavahetused, pühad) jäetakse vahele
  • Ülesannete kestus muutub automaatselt, kui tööaeg muutub
Automaatse ajastusega ülesanded arvutatakse automaatselt kalendri põhjal.
